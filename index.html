Features

Demo

No Set up required

ReactION requires NO modification to your codebase, so using it is as simple as installing our VS Code Extension

Works with All your favorite libraries

React-Sight supports React 14.7+ (including Fiber), React Router v4, React Redux, and more

Built for Speed

React-Sight is light-weight and unobtrusive, and will show you how your Application is working

See only what you want to see

Built in filters allow you to hide elements you don't want to see, allowing you focus only on the components that you've written

Setup


Enable React DevTools

ReactION requires VS Code installation

Add React Sight to Chrome

Install React Sight from the Chrome web store.
For local projects, check 'Allow access to file URLs' on the extensions page

Run your Application

Open your React application, or open (almost!) any website running React!

Open ReactION

Open the VS code for ReactION
